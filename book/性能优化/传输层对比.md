## 场景选择建议

| 场景 | 推荐传输层 | 原因 |
|------|------------|------|
| 本地开发 | Stdio | 最低延迟，配置简单 |
| 简单远程服务 | HTTP | 兼容性好，易于调试 |
| 实时更新 | SSE | 支持服务端推送 |
| 高并发生产 | Streamable HTTP | 性能最优 |

---

## 本章小结

1. **Stdio**——最适合本地开发
2. **HTTP**——通用性强
3. **SSE**——支持实时推送
4. **Streamable HTTP**——生产环境最优
