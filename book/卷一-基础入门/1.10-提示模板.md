# 1.10 提示模板的使用

> 理解 MCP 提示模板的概念和使用方法

---

## 什么是提示模板（Prompts）

提示模板是 MCP 中的**可复用提示**，可以快速执行常见任务。

```
┌─────────────────────────────────────────────────────────────────┐
│                      提示模板的价值                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  场景：                                                         │
│  你经常需要让 AI 帮助审查代码                                    │
│                                                                 │
│  ❌ 每次都要输入完整的审查提示                                   │
│  ✓ 使用提示模板，一键触发                                       │
│                                                                 │
│  /code_review diff=xxx                                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

## 提示模板定义

```json
{
  "name": "prompt_name",
  "description": "模板描述",
  "arguments": [
    {
      "name": "arg_name",
      "description": "参数描述",
      "required": true
    }
  ]
}
```

---

## 提示模板示例

```python
from fastmcp import FastMCP

app = FastMCP("Prompt Templates")

@app.prompt()
def code_review(diff: str) -> str:
    """代码审查提示模板"""
    return f"""请审查以下代码变更：

```diff
{diff}
```

请关注：
1. 代码质量问题
2. 安全漏洞
3. 性能优化
4. 最佳实践
"""

@app.prompt()
def write_tests(function_code: str) -> str:
    """生成测试用例的提示模板"""
    return f"""请为以下函数生成单元测试：

{function_code}

要求：
1. 使用 pytest 框架
2. 覆盖主要场景
3. 包含边界条件测试
"""
```

---

## 使用提示模板

在 Claude Desktop 中，输入：

```
/code_review diff="+def hello(): print('hello')"
```

或者：

```
/write_tests function_code="def add(a, b): return a + b"
```
