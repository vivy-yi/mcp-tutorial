# 1.7 第一个 MCP 服务器实战

> 手把手构建你的第一个 MCP 服务器

---

## 使用 FastMCP 创建服务器

### 安装 FastMCP

```bash
pip install fastmcp
```

### 创建服务器代码

```python
from fastmcp import FastMCP

# 创建 MCP 应用
app = FastMCP("My First Server")

# 定义工具
@app.tool()
def greet(name: str) -> str:
    """向用户发送问候"""
    return f"你好，{name}！欢迎使用 MCP！"

@app.tool()
def add(a: int, b: int) -> int:
    """计算两个数的和"""
    return a + b

# 运行服务器
if __name__ == "__main__":
    app.run()
```

---

## 配置到 Claude Desktop

```json
{
  "mcpServers": {
    "my-first-server": {
      "command": "python",
      "args": ["/path/to/your/server.py"]
    }
  }
}
```
