# 1.5 第一个 MCP：安装与配置

> 本章将手把手教你安装和配置第一个 MCP，带你迈出 MCP 开发的第一步

---

## 环境准备

在开始之前，确保你的系统满足以下要求：

| 要求 | 说明 |
|------|------|
| **操作系统** | macOS 12+ / Windows 10+ / Ubuntu 20.04+ |
| **Node.js** | 18+ (使用 npx) |
| **Python** | 3.10+ (可选，用于 Python MCP) |
| **网络** | 能够访问 npm 仓库 |

---

## 步骤一：安装 Claude Desktop

Claude Desktop 是体验 MCP 最简单的方式。

### macOS 安装

```bash
# 方法一：Homebrew
brew install --cask anthropic

# 方法二：手动下载
# 访问 https://claude.ai/desktop 下载安装包
```

### Windows 安装

```powershell
# 手动下载
# 访问 https://claude.ai/desktop 下载 .exe 安装包
```

### Linux 安装

```bash
# 下载 AppImage
wget https://claude.ai/desktop/latest/linux/Claude-Setup-x64.AppImage

# 添加执行权限
chmod +x Claude-Setup-x64.AppImage

# 运行
./Claude-Setup-x64.AppImage
```

---

## 步骤二：配置 MCP 服务器

### 打开配置文件

Claude Desktop 的配置文件位于：

```
macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json
Linux: ~/.config/Claude/claude_desktop_config.json
```

### 创建第一个 MCP 配置

```json
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-filesystem",
        "/Users/yourname/Desktop"
      ]
    }
  }
}
```

### 验证安装

1. 重启 Claude Desktop
2. 询问 AI："我的桌面有什么文件？"
3. 如果 AI 能够回答，说明 MCP 配置成功！

## 步骤三：添加更多 MCP 服务器

### GitHub MCP 服务器

```json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_TOKEN": "ghp_xxxxxxxxxxxx"
      }
    }
  }
}
```

> **注意**：你需要创建一个 GitHub Personal Access Token，访问 https://github.com/settings/tokens

### SQLite MCP 服务器

```json
{
  "mcpServers": {
    "sqlite": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-sqlite", "--db-path", "./data.db"]
    }
  }
}
```

### Brave 搜索 MCP 服务器

```json
{
  "mcpServers": {
    "brave-search": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-brave-search"],
      "env": {
        "BRAVE_API_KEY": "your-brave-api-key"
      }
    }
  }
}
```

## 步骤四：多服务器配置

你可以在配置文件中添加多个 MCP 服务器：

```json
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/Users/yourname/Desktop"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_TOKEN": "ghp_xxx"
      }
    },
    "brave-search": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-brave-search"],
      "env": {
        "BRAVE_API_KEY": "xxx"
      }
    }
  }
}
```

## 常见问题排查

### 问题一：MCP 服务器不工作

**症状**：Claude Desktop 无法调用 MCP 工具

**排查步骤**：

1. 检查配置文件语法是否正确
2. 确认 npx 可用：`npx --version`
3. 手动运行服务器命令测试
4. 查看 Claude Desktop 日志

### 问题二：权限被拒绝

**症状**：访问文件时报错

**解决**：确保配置文件中的路径正确，且有访问权限

### 问题三：Token 失效

**症状**：GitHub API 调用失败

**解决**：检查 GITHUB_TOKEN 是否过期，重新生成

## 进阶：使用 Python MCP

如果你更喜欢使用 Python 开发 MCP，可以使用 FastMCP 框架：

### 安装 FastMCP

```bash
pip install fastmcp
```

### 创建自定义 MCP 服务器

```python
from fastmcp import FastMCP

app = FastMCP("My MCP Server")

@app.tool()
def calculate(a: int, b: int) -> int:
    """计算两个数的和"""
    return a + b

if __name__ == "__main__":
    app.run()
```

### 配置 Python MCP

```json
{
  "mcpServers": {
    "my-mcp": {
      "command": "python",
      "args": ["/path/to/your/mcp_server.py"]
    }
  }
}
```
