---

## 配置文件结构

```json
{
  "mcpServers": {
    "server-name": {
      "command": "执行命令",
      "args": ["参数1", "参数2"],
      "env": {
        "环境变量": "值"
      },
      "config": {
        "自定义配置": "值"
      }
    }
  }
}
```

---

## 字段详解

| 字段 | 必填 | 说明 |
|------|------|------|
| `command` | ✅ | 执行的命令 (npx, python, node 等) |
| `args` | ✅ | 命令参数数组 |
| `env` | ❌ | 环境变量 |
| `config` | ❌ | 自定义配置 (传递给服务器) |

---

## 高级配置示例

### 配置传递自定义参数

```json
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-filesystem",
        "--allowed-directory",
        "/Users/yourname/projects"
      ]
    }
  }
}
```

### 多环境变量配置

```json
{
  "mcpServers": {
    "github": {
      "command": "python",
      "args": ["./mcp_github/server.py"],
      "env": {
        "GITHUB_TOKEN": "ghp_xxx",
        "GITHUB_ORG": "my-org",
        "LOG_LEVEL": "debug"
      }
    }
  }
}
```
