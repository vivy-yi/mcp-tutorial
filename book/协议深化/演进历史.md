## 版本详细对比

### v1.0 (2024年11月) - 初始版本

**发布背景**：
- Anthropic 正式发布 MCP 协议
- 与 Claude Desktop 深度集成

**核心功能**：
```json
{
  "protocolVersion": "1.0",
  "capabilities": {
    "tools": {},
    "resources": {},
    "prompts": {}
  },
  "serverInfo": {
    "name": "server-name",
    "version": "1.0.0"
  }
}
```

**局限性**：
- 只支持 Stdio 传输
- 缺乏高级认证机制
- 无流式响应支持

---

### v1.1 (2025年1月) - 完善版本

**新增功能**：

| 功能 | 说明 |
|------|------|
| HTTP 传输 | 支持远程 MCP 服务器 |
| SSE | 服务端推送支持 |
| 能力协商 | 客户端/服务端能力交换 |
| 改进的错误处理 | 标准化的错误格式 |

**协议变更**：

```json
{
  "protocolVersion": "1.1",
  "capabilities": {
    "tools": {},
    "resources": {},
    "prompts": {},
    "logging": {}
  }
}
```

---

### v1.2 (2025年3月) - 增强版本

**新增功能**：

| 功能 | 说明 |
|------|------|
| Streamable HTTP | 更高效的传输方式 |
| 采样 (Sampling) | 服务端可请求 AI 采样 |
| 根对象 (Roots) | 资源根目录定义 |
| 展开 (Elication) | 服务端请求用户输入 |

**采样示例**：

```json
{
  "method": "sampling/createMessage",
  "params": {
    "messages": [...],
    "systemPrompt": "你是一个有帮助的助手",
    "maxTokens": 1024
  }
}
```

## 迁移指南

### 从 v1.0 迁移到 v1.1

```python
# v1.0 客户端
from mcp import ClientSession, StdioServerParameters

# v1.1 客户端 (兼容旧版本)
from mcp import ClientSession
from mcp.client.http import HttpServerParameters

# 添加 HTTP 支持
server_params = HttpServerParameters(
    host="localhost",
    port=8080
)
```

### 从 v1.1 迁移到 v1.2

```python
# 添加新功能支持
from mcp.types import SamplingMessage

# 使用采样功能
async def handle_sampling_request(session, request):
    result = await session.create_sampling_message(
        messages=request.messages,
        systemPrompt="你是一个有帮助的助手"
    )
    return result
```
